可嵌入数据

在json代码块使用数据模型目录下的json数据 example.json:

{
  // sss
  "x": "sss", //asdjf;jk;
  "items": "xxxx"
}

在文档中的写法:

\`\`\`json //这里是转义了
{
    "example": "{{example.json}}"
}
\`\`\`

注意"{{filename}}", filename为models里的文件名，key必须用"{{}}"包裹 最终展现为：

{
    "example": {
      //可以使用行注释
      "key": "sss",
      "arr": [ {
        //可以使用行注释
        "a": "aaa"
      }, {
        //可以使用行注释
        "a": "aaa"
      }, {
        //可以使用行注释
        "a": "aaa"
      }]
    }
}
