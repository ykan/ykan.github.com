# 自定义的Markdown

## 可嵌入数据
在`json`代码块使用数据模型目录下的json数据
`example.json`:
```
{
  // sss
  "x": "sss", //asdjf;jk;
  "items": "xxxx"
}
```
在文档中的写法:
```
\`\`\`json //这里是转义了
{
    "example": "{{example.json}}"
}
\`\`\`
```
> 注意`"{{filename}}"`, filename为`models`里的文件名，key必须用`"{{}}"`包裹
最终展现为：

```json
{
    "example": "{{example.json}}"
}
```

## 插入导航

```
\`\`\`nav-list
commands
\`\`\`
```
`渲染结果`：
```nav-list
nodejs
```
